# MJ-Tribute-Concert-Booking-System
A full-stack concert booking platform built for managing Michael Jackson tribute concert events, ticket bookings, user authentication, and admin event management.

# Project Overview

The Michael Jackson Tribute Concert Booking System is a web-based application that allows users to:
Register and log into the system
Browse upcoming concert events
Book concert tickets
Manage bookings
View a personal dashboard
Allow administrators to manage events and users

The system follows the MVC (Model-View-Controller) architecture and uses MongoDB Atlas as the cloud database.

# Setup:

1. Extract the project folder.

2. Open the project in Visual Studio Code.

3. Open the terminal.

4. Install all required dependencies:

npm install

5. Run the application:
  node app.js

6. http://localhost:9100


# Reflection

Working with new technologies within a limited timeframe proved challenging. Despite encountering issues such as virtual machine compatibility with Express.js and firewall restrictions affecting MongoDB Atlas connectivity, the team successfully collaborated to overcome these obstacles and complete the project. The experience improved our understanding of full-stack web development, teamwork, debugging, and cloud database integration.
